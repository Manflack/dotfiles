# My dotfiles 

## 🚀 Installation

Using wget
```bash
bash <(wget -qO- https://raw.githubusercontent.com/franciscoruizar/dotfiles/HEAD/installer)
```

Using curl
```bash
bash <(curl -s https://raw.githubusercontent.com/franciscoruizar/dotfiles/HEAD/installer)
```

## Switch to zsh shell
Using curl
```bash
chsh -s $(which zsh)
```
